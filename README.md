# GetHiredWithAI

**Get hired faster with AI-powered resume optimization**

Upload your resume once. Get personalized versions for every job. 
GetHiredWithAI uses GPT-4 to rewrite your resume, matching job descriptions 
while maintaining authenticity.

## Features
AI Resume Rewriter - Customized for each job
Intelligent Job Matching - 80%+ accuracy
Skills & Experience Analysis
Secure & Private
One-click Apply Tracking

## Tech Stack
- Streamlit (UI)
- OpenAI GPT-4 (AI)
- Supabase (Database)
- Railway (Hosting)
- NameCheap (Domain)
